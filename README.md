Gym Buddy is a menu-driven Java console application designed for gym enthusiasts to track basic fitness metrics and get simple workout guidance. The project is built using core Java concepts and focuses on clean logic and modular design.

---

## Features

- **DOTS Strength Score Calculator**
  - Supports both male and female lifters
  - Classifies strength level from beginner to world-class

- **Workout Plan Selector**
  - Push Day
  - Pull Day
  - Leg Day

- **Workout Logging**
  - Logs workout duration
  - Estimates calories burned

- **BMI Calculator**
  - Calculates Body Mass Index
  - Categorizes BMI based on standard ranges
